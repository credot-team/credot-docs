## 建立 javascript 專案

```sh
npx create-react-app my-app
```

## 建立 typescript 專案

```sh
npx create-react-app my-app --template typescript
```

## 執行專案

```sh
cd my-app
yarn start
```

## 可用指令

### yarn start

```sh
yarn start
```

### yarn test

```sh
yarn test
```

### yarn build

```sh
yarn build
```

### yarn eject

```sh
yarn eject
```

## 設定 Proxy

```json
 "proxy": "http://localhost:8080",
```
