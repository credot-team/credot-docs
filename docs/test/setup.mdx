## 常用 libraray

| Type          | Scope                                                                                                             | Tooling                                                    |
| ------------- | ----------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| `Unit`        | Tests the functions and methods of an application.                                                                | `Ava`<br />`Jasmine`<br />`Jest`<br />`Karma`<br />`Mocha` |
| `Integration` | Tests Interactions between units.                                                                                 | `Ava`<br />`Jest`<br />`Testing Library`                   |
| `End-to-end`  | Tests user interactions in a real-life browser by providing it instructions for what to do and expected outcomes. | `Cypress`<br />`Puppeteer`                                 |

## 設定

### 安裝 Jest

```bash
yarn add -D jest @types/jest
```

### 安裝 Puppeteer

```bash
yarn add -D puppeteer
```
